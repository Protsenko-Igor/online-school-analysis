# Дашборды и визуализация

В этом проекте использовалась платформа **Yandex DataLens** для создания интерактивных дашбордов.

## Структура дашборда

### Основной дашборд: "Отчет онлайн школы по психологии (2023)"
Состоит из двух ключевых вкладок:

#### 1. Вкладка "Выручка и расходы"
![Воронка продаж](sales_funnel_tab.png)
**Цель:** Анализ эффективности маркетинговых каналов
**Элементы:**
- Воронка продаж (клики → лиды → продажи) с конверсиями в разрезе источников и кампаний
- Линейные графики динамики CPO и количества продаж по неделям
- Селекторы для фильтрации по дате, источнику, кампании

#### 2. Вкладка "Вовлеченность учеников (когорты)"
![Когортный анализ](retention_tab.png)
**Цель:** Анализ удержания и вовлеченности студентов
**Элементы:**
- Таблица с Heatmap (тепловая карта) Retention Rate по месячным когортам
- Стобчатые диаграммы с динамикой прохождения уроков и дз по 10 неделям
- Селекторы для выбора продукта и конкретной когорты

## Используемые типы чартов
- **Линейные графики** (Line charts) - для временных рядов
- **Столбчатые диаграммы** (Bar charts) - для сравнения категорий
- **Таблицы** (Tables) - для детализированных данных
- **Heatmaps** - для когортного анализа retention
- **Показатели** (Indicators) - для KPI

## Источники данных в DataLens
Подключены два основных датасета:
1. `sales_funnel_df` - агрегированная воронка продаж
2. `retention_df` - агрегированные данные по когортам
